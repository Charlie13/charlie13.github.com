Youtube의 MIT 파이썬 강좌

  • MIT Python Course: https://www.youtube.com/playlist?list=PL57FCE46F714A03BC
  • 전체: 24개 강좌로 구성 (동영상 평균 60분)
  • 2016.03.29현재 18편까지 한글 자막 제공

Microsoft Virtual Academy

  • [https://mva.microsoft.com/search/SearchResults.aspx#!q=python&lang=1042][https://mva.microsoft.com/search/SearchResults.aspx#!q=python&lang=1042]
