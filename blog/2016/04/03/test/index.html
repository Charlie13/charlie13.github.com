      1.1 기계학습이란 무엇인가?

  • 기계학습은 데이터를 정보로 변환하는 것

  • 기계학습은 통계학을 사용함

  • 육체 노동자들이 지식 노동자로 전환하면서 생긴일

    • 모호한 일들이 일반화 됨
    • 이익 최대화
    • 손실 최소화

    • 최상의 영업 방법 찾기

      1.2 주요 용어

  • 1.3 기계학습의 주요 기술

  • Supervised Learning

    • 알고리즘이 무엇을 예측할 것인지를 제공
    • 유형: Classification, Regerssion

  • NoSupervised Learning

    • 비지도학습
    • 분류 항목 표시나 목적 변수가 없음
    • 유형
    • clustering: 군집화

    • Density Estimation: 밀도 추적

      알고지즘 정리

  • Supervised Learning

    • Classification (분류)
    • Regression (회귀)
    • K-nearest Neighbors
    • Linear
    • Naive Bayes
    • Locally Weighted Linear
    • Support Vector Machine
    • Ridge
    • Decision Tree
    • Lasso

  • UnSupervised Learning

    • Clustering
    • Density Estimation
    • K-Mean
    • Expection maximization
    • DBSCAN
    • Parzen Window
