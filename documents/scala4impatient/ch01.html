Ch01. The Basics

주요 내용 정리

  • import scala.math._
  • import 문에서 scala로 시작하는 패키지는 scala 생략 가능
    • 다음 import는 동일한 결과
    • import scala.math._
    • import math._
  • static 메서드가 없음
    • Companion Object에 자바이 static 메서드 정의 가능
    • 인자가 없는 스칼라 메서드는 일반적으로 괄호를 사용하지 않는다.
  • apply 메서드
    • apply메소드는 다음과 같은 상황헤서 묵시적으로 호출됨
    • "Hello"(4)
    • BigInt("1234567890")
      • apply 메서드는 Companion Object에 정의됨
      • new 연산자 없이 객체 생성

Exercises

1. 스칼라 REPL에서 3을 입력하고 Tab키를 입력할 것. 어떤 메서드를 볼 수 있나?

scala> 3.

2. 스칼라 REPL에서 3의 제곱근을 계산하고 그 값을 제곱하라. 결과가 3과 얼마나 차이가 나는가?

scala> import math._
import math._

scala> sqrt(3)
res2: Double = 1.7320508075688772

scala> pow(res2, 2)
res3: Double = 2.9999999999999996

scala> 3 -  res3
res4: Double = 4.440892098500626E-16
