Problem NintyNine for Scala

prolog 학습 자료인 “P-99: Ninety-Nine Prolog Problems”의 스칼라 버전 입니다. “P-99: Ninety-Nine Prolog Problems”는 Prolog의 학습을 지원하기 위한 연습문제 성격입니다. Prolog버전의 연습문제를 Scala 버전으로 변경하여 제공하는 내용을 정리하는 페이지 입니다. “P-99: Ninety-Nine Prolog Problems”은 홈페이지는 다음과 같습니다.

  • https://sites.google.com/site/prologsite/prolog-problems

“P-99: Ninety-Nine Prolog Problems”은 7개 카테고리로 구성되며, 스칼라 버전의 변환은 aperiodic.net을 참조하여 정리하였습니다.

해당 문제의 답안은 문제 번호에 설정된 링크로 확인 할 수 있습니다.

  • 카테고리 구성
    • List
    • Arithmetic
    • Logic and Codes
    • Binary Trees
    • Multiway Trees
    • Graphs
    • Miscellaneous

1. List

  • S-01: 리스트의 마지막 엘리먼트를 구하시오?
    • 예제 코드
scala> last(List(1, 1, 2, 3, 5, 8))
res0: Int = 8
  • S-02: 리스트의 끝에서 두번째 엘리먼트를 구하시오?
    • 예제 코드
scala> penultimate(List(1, 1, 2, 3, 5, 8))
res0: Int = 5
  • S-03: 리스트의 k번째 엘리먼트를 구하시오?
    • 리스트의 첫 번째 엘리먼트는 0.
    • 예제 코드
scala> nth(2, List(1, 1, 2, 3, 5, 8))
res0: Int = 2
  • S-04: 리스트의 엔리먼트 사이즈를 구하시오?
    • 예제 코드
scala> length(List(1, 1, 2, 3, 5, 8))
res0: Int = 6
  • S-05: 리스트를 리버스로 변환하시오?
    • 예제 코드
scala> reverse(List(1, 1, 2, 3, 5, 8))
res0: List[Int] = List(8, 5, 3, 2, 1, 1)
